Instuctions:

Description:
Created an AI in Java to solve the Hangman game within 3 errors.
run an algorithm with the alphabet frequencies of a large dictionary to determine the best guess and make the guess.

External dependency:
org.json jar
Download from:
https://code.google.com/p/org-json-java/downloads/list

First go the the src directory of the project

How to compile?
javac -cp org.json.jar HangmanGame.java Hangman.java Game.java GetInfo.java

How to run?
java -cp .:./org.json.jar HangmanGame

You can view the accuracy rate in the terminal
Success rate ranges from 30%-40%.

Users can terminate the application using Control+C.
